# Simples API utilizando NodeJS e MongoDB

## Instalação

* npm install

## Configuração

Configure o arquivo **config.js** localizado na pasta config com a porta desejada para rodar o servidor web e também a **uri** do MongoDB

## Rodando o Projeto

* npm start
